# Student Management System GUI

A professional beginner project using Python, Tkinter, and SQLite.

## Features
- Add Student
- View Students in Table
- Update Student Details
- Delete Student
- Clear Input Fields

## Technologies Used
- Python
- Tkinter
- SQLite

## How to Run
1. Install Python
2. Save file as main.py
3. Run command:

python main.py
